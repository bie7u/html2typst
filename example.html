Text Formatting

This paragraph demonstrates bold text, italic text, underlined text, strikethrough text, and highlighted text.

You can also use superscript and subscript for scientific notation like E=mc2 or chemical formulas like H2O.

Code Examples

Inline code can be written like this: print("Hello, World!")

Code blocks are formatted as follows:

def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)
