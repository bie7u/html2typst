Getting Started with Typst

Published on January 15, 2024

Introduction

Typst is a new markup-based typesetting system that aims to be as powerful as LaTeX while being much easier to learn and use.

Why Typst?

  • Clean, readable syntax
  • Fast compilation
  • Modern tooling
  • Great error messages

Basic Example

Here's a simple Typst document:

#set document(title: "My Paper")
#set page(numbering: "1")

= Introduction

This is a paragraph.

Learn More

Visit the official Typst website to learn more and try the online editor.

Typst makes it easy to create beautiful documents without the complexity of LaTeX.
